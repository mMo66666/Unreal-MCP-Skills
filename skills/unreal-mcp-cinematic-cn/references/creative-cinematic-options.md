# 创意授权下的电影化候选

仅在设计师明确要求“补分镜 / 你来发挥 / 给创意提案”时读取。以下内容提供判断维度，不是镜头套餐、固定镜数或风格公式。

## 先确定

- 观众此刻必须读到的主体、动作、信息或情绪。
- 观众站位：旁观、贴近角色、主观感知，或拥有角色不知道的信息。
- 当前镜头新增什么；没有信息、情绪或节奏价值的镜头不主动增加。
- 动作、视线、屏幕方向和空间轴线如何连接前后镜头。

## 镜头候选

- 景别、机位、运动和焦距由主体阅读与情绪距离共同决定，不按情绪标签自动套镜头。
- 推、拉、跟随、环绕、升降、静镜、反应或留白都只是候选；每个选择应能说明画面职责。
- 情绪转折可改变视觉强度、运动、构图或节奏，但变化不必同时发生。
- 低机位、高机位、遮挡、中心构图、留白和浅景深会改变观众判断，使用前确认没有抢错主体或损失动作信息。

## 组接与节奏

- 切点可来自动作、视线、结果、反应、声音或信息完成时刻。
- 连续性服务理解；有意跳轴、方向反转或信息限制必须有叙事目的。
- 时长按动作关键姿势、信息量、情绪变化和总时长决定；均分或非均分都只是候选，不固定镜数。
- 高潮、反应和余韵是否存在由设计师目标决定，不自动补齐完整戏剧模板。

## 输出边界

- 把 AI 新增内容标为提案或执行假设，不伪装成设计师原意。
- 设计师确认方案后，再转成 `storyboard-to-sequencer-spec.md` 和 UE 执行字段。
- 创意参考不得覆盖资产安全、帧契约、CameraCut、控制权和证据边界。
